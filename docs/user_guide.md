# User Guide: Private AI Agents Portal

Welcome to the Private AI Agents Portal! This guide helps you navigate and use the portal.

---

## 1. Registering for Access

1. Open the portal URL in your web browser.
2. Click **Continue with Google** and sign in using your preferred Google account.
3. If this is your first time visiting, you will be prompted to enter an **Invitation Code**.
4. Enter the code provided by the administrator and click **Verify & Continue**.
5. Your account is now submitted for admin review. Once approved, the page will refresh automatically and load the catalog.

---

## 2. Navigating the Catalog

The catalog organizes AI Agents into four categories:
* **🧠 Plan:** Strategy, roadmaps, idea validation, and preparation tools.
* **⚙️ Do:** Development, coding, reverse engineering, and action generation.
* **🔍 Check:** Document validators, testers, UI/UX audit, and code quality checkers.
* **🚀 Act:** Next step strategies, Suno musical prompt compositions, astrological decision engines, and workload prioritize coaches.

---

## 3. Search, Sort, and Category Filters

* **Search:** Type in the search bar. This filters agents in real-time, matching words in both the **agent name** and the **description**.
* **Sort:** Toggle the **Sort** button to arrange listings alphabetically (A-Z or Z-A).
* **Category Filters:** Filter by category (e.g. Plan) using the dropdown tab to clean up the screen.

---

## 4. Desktop vs Mobile Features

* **Desktop (Mouse):** Hover your cursor over any agent link. A tooltip box will appear displaying the agent's description.
* **Mobile (Touch):** Tap the small **Info (i)** icon next to the agent link to pull up a bottom-sheet detailing the agent.

---

## 5. Favoriting Agents

Click the **Star** icon next to any agent to add it to your personal favorites. You can toggle the **Favorites Only** button at the top to display only your selected agents, creating a personalized workspace.
