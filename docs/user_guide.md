# User Guide: Private AI Agents Portal

Welcome to the Private AI Agents Portal! This guide helps you navigate and use the directory.

---

## 1. Entering the Portal

To access the directory, you must use a cryptographic access pass issued by the portal administrator:
1. Open the portal URL in your web browser.
2. In the Login form, enter your **Email Address**.
3. In the **Access Pass** field, copy and paste the cryptographic pass token shared with you by the administrator.
4. Click **Access Portal** to log in.

---

## 2. Navigating the Catalog

The directory organizes AI Agents into four columns:
* **🧠 Plan:** Strategy, validation, planning, and preparation tools.
* **⚙️ Do:** Development, coding, reverse engineering, and actions generators.
* **🔍 Check:** Validators, UI/UX audit assistants, and code testers.
* **🚀 Act:** Astrological engines, musical compositions, prioritization coaches, and advanced strategies.

---

## 3. Search and Sort Tools

* **Search:** Type in the **Search agents...** bar. The portal filters agents in real-time, matching queries against agent names and descriptions.
* **Sort:** Click the **Sort: A-Z** (or **Sort: Z-A**) button to rearrange lists alphabetically in ascending or descending order.

---

## 4. Hover Descriptions (Tooltips)

* Hover your mouse cursor over any agent link.
* A tooltip box will appear displaying the agent's full description.
