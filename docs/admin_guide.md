# Administrator Guide: Private AI Agents Portal

This guide provides instructions to perform daily administrative operations on the Private AI Agents Portal.

---

## 1. Accessing the Administration Console

1. Open the portal login page in your browser.
2. Enter your administrator email: **`rajajeevankumar@gmail.com`**.
3. Enter your master administrator passphrase in the Access Pass field (this is configured via `VITE_ADMIN_SECRET` on Vercel).
4. Click **Access Portal**. You will be logged in as a Super Admin and redirected to `/admin`.

---

## 2. Issuing Access Passes for Guests

Guests require a cryptographically signed token pass to browse the catalog:
1. Navigate to the **Administration Portal** page.
2. Select the **Access Passes** tab.
3. Under **Issue Access Pass**:
   * **Guest Email:** Enter the guest's email address (e.g. `guest@example.com`).
   * **Pass Duration:** Select pass validity (`1 Month`, `3 Months`, or `1 Year`).
4. Click **Create Access Pass**.
5. The portal computes the HMAC-SHA256 signature and renders the token string.
6. Click **Copy Pass Token** and share it with the guest.

---

## 3. Revoking and Tracking Passes

* **Tracking:** The **Issued Access Passes** table shows all tokens generated by the administrator.
* **Revoking:** Click **Revoke** next to any guest email record to remove their entry from your administration tracker. This action is logged in the local audit logs.

---

## 4. Viewing Local Audit Logs

Select the **Audit Logs** tab on the Administration Portal page.
* This tab lists all token generation and revocation actions performed by the administrator.
* Audit data is saved in local browser storage (`local_audit_logs`).

---

## 5. Managing the Catalog (CRUD)

Super Admins can edit the directory registry directly from the Catalog page:
1. Return to the Catalog page.
2. Click the floating **+ (FAB)** button in the bottom right corner.
3. **Add New Agent:** Fill in Name, URL, Description, select a Category (`Plan`, `Do`, `Check`, `Act`), and click **Save Agent**.
4. **Edit Agent:** Locate the agent under the **Active Registry** section in the modal, click **Edit**, adjust the details, and click **Save Agent**.
5. **Delete Agent:** Click **Delete** next to any agent in the Active Registry listing to remove it from the directory catalog.

---

## 6. Backups & Offline Portability

The floating **+ (FAB)** modal contains tools to export and backup directories:
* **Export JSON:** Downloads your current merged catalog registry as a standard `custom_gpt_backup.json` file.
* **Import JSON:** Allows loading an exported JSON backup to reconstruct/merge directory items.
* **Embed & Save Standalone File:** Compiles the entire catalog database into a portable, self-contained single-file HTML page. You can run this file offline in any browser to access the directory without a network connection.
